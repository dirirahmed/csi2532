# eHotels Database

## SQL files
- `ehotels_schema.sql` : creates the database and tables
- `ehotels_inserts.sql` : inserts sample data
- `ehotels_advanced.sql` : creates views, indexes, and triggers
- `ehotels_queries.sql` : contains sample queries for testing

## Execution order
Run the files in this order:

1. `ehotels_schema.sql`
2. `ehotels_inserts.sql`
3. `ehotels_advanced.sql`
4. `ehotels_queries.sql`

## Notes
- Database name: `ehotels`
- Tested using MySQL Workbench
